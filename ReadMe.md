## How to build
To build this project, there are three prerequisites:
* `cmake -DCMAKE_BUILD_TYPE=Release /path/to/src`
* OpenCV Library
* Boost C++ Library

### OpenCV Installation

First download the latest OpenCV from its official website. 

## OS X OpenMP Notice
* `brew install gcc`
* `brew install boost --cc=gcc-6`
